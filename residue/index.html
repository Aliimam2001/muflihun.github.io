Introduction

Residue is efficient real-time centralized logging server. It has been built with speed and security in focus. Applications can send log messages and the server will write logs to log file using default or specific configurations.

Residue uses Easylogging++ under the hood, that itself is a feature-rich library. Server admin can provide access to their clients to update their loggers (configuration, access codes etc)
